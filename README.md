# discord-ship-bot
Discord bot, in Python, to create fun random romance ships within a server

HOW TO USE
1. Download repository
2. Extract the entire .zip file into a folder
3. Open the bot.py file with a text editor, replace "MY TOKEN" with your token created on the Discord Developers platform
4. In the Discord Developers platform, grant the following permissions: - View channels - Send messages - Send messages in threads - Use slash commands
5. Run bot.py
